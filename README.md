# Patriots

Patriots 是一个基于机器学习、数据挖掘、知识图谱的个人大数据管理平台

"Patriots" is a Personal Big Data Management Platform for Web / Mobile / WechatApp based on data mining, machine learning & knowledge graph techniques.

## 技术栈

+ 后端
    + Go 1.8
    + Beego 1.8.0 (Bee 1.8.1)
    + python 2.7
    + tensorflow 1.0.1
    + keras 2.0.1
+ 前端
    + vue.js
    + element.ui - web
    + vux - mobile
+ Python 包
    + gensim 0.13.3
    + jieba 0.38
    + matplotlib 2.0.0
    + numpy 1.11.2
    + pandas 0.18.1
    + Pattern 2.6
    + Pillow 4.0.0
    + scikit-learn 0.18.1
