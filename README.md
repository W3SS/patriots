# Patriots

一个集机器学习、数据挖掘等各类人工智能实践的 web 管理系统

Data Mining & Machine Learning / Deep Learning Platform with Web UI

## 技术栈

+ 后端
    + Go 1.8
    + Beego 1.8.0 (Bee 1.8.1)
    + python 2.7
    + tensorflow 1.0.1
    + keras 2.0.1
+ 前端
    + vue.js
    + element.ui
+ Python 包
    + gensim 0.13.3
    + jieba 0.38
    + matplotlib 2.0.0
    + numpy 1.11.2
    + pandas 0.18.1
    + Pattern 2.6
    + Pillow 4.0.0
    + scikit-learn 0.18.1
